# ThermoCool Landing Page

A professional, responsive landing page for ThermoCool HVAC Services showcasing company information, services, and contact details.

## Project Overview

This is a static HTML/CSS landing page built with modern web standards, featuring:

- Semantic HTML5 structure for accessibility and SEO
- Responsive design using CSS Grid and Flexbox
- Mobile-first approach supporting desktop, tablet, and mobile devices
- BEM (Block Element Modifier) CSS methodology for maintainable styling
- Professional typography using Google Fonts
- Optimized for modern browsers (Chrome, Firefox, Safari, Edge)

## Technology Stack

- **HTML5**: Semantic markup with proper document structure
- **CSS3**: Modern styling with Grid, Flexbox, and custom properties
- **Google Fonts**: Professional typography (Roboto family)
- **Python 3**: Local development server
